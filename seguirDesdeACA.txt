package.json SCRIPTS:

al comando que esta en json se le puede agregar la "-w" al final para hacerlo watcher
node-sass sass/main.scss css/style.css -w

con -w funciona como un watcher y podes trabajar sin tener que compilar a cada rato.

IMPORTANTE:

para compilar sass usar el siguiente comando:

npm run compile:sass

para correr el live server, abrir otro tab en la terminal y escribir

live-server
